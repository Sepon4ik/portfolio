# Фоновая музыка (`/audio/ambience.mp3`)

Сейчас в проекте лежит **ambience.mp3** (спокойная космическая/эмбиент музыка). Плеер на сайте по умолчанию берёт этот файл.

1. Чтобы заменить трек — перезапишите **`ambience.mp3`** или задайте другой путь через `NEXT_PUBLIC_AMBIENCE_AUDIO_SRC` в `.env`.
2. Соблюдайте **лицензию** источника (если трек со стока — укажите автора/ссылку на странице «О сайте» или в футере при публикации в интернете).
3. Воспроизведение: кнопка **слева внизу** (браузеры без клика автозвук обычно не включают).

**Идеи бесплатных треков:** [Pixabay Music](https://pixabay.com/music/), [Free Music Archive](https://freemusicarchive.org/), [Uppbeat](https://uppbeat.io/).
